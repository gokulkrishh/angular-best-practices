Always use [] notation for listing dependencies



//Bad
angularApp.controller(function($scope) {
	//Write something here
});

//Good
angularApp.controller(['$scope', function($scope) { 
	//Write something here
}]);

Reason:

Angular get its dependencies from names of arguments, during minification the arguments in the function are converted to a, b, c ... So giving dependencies in [] notation helps angular to map its dependencies.
