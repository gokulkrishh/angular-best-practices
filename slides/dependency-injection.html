Dependency Injection

Dependency Injection (DI) is very powerful concept.

  1. DI is passing of a dependency (Service) to a client (Controller), rather than allowing a client to build or find the service.

//Creating app module
var app = angular.module('appName', []);

//Creating a controller
app.controller('mainCtrl', ['$scope', myService', function ($scope, myService) {
  myService.hello('Gokul'); //will alert Hello Gokul
}]);

//Creating a service
app.service('myService', ['$http', function ($http) {
  this.hello = function (name) {
    alert('Hello ' + name);
  };
}]);
